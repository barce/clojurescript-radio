Introduction to ClojureScript

Why Clojure?

  • Stable, consistent syntax
  • Large core library
  • Persistent Data Structures
  • Language-level state management
  • Flexible syntax (future-proof)
  • Awesome community

Why ClojureScript?

  • Clojure compiled to JavaScript
  • Leverages the best of Google Closure toolchain
  • Build system is part of the language
  • Flexible syntax (HTML, async)
  • Stable!
