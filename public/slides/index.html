Introduction to ClojureScript

What is Clojure?

  • Belongs to the lisp family
  • Runs on the JVM
  • Designed for concurrency
  • Community driven development process

Why Clojure?

  • Simple and practical
  • Stable, flexible syntax (future-proof)
  • Powerful core library
  • Persistent Data Structures as default
  • Awesome community
Great video on Clojure:
Simple Made Easy - Rich Hickey

What is ClojureScript?

  • Clojure compiled to JavaScript
  • Functional language
  • Dynamically typed
  • Lisp syntax
(def pi 3.14)

(defn by-id [id]
  (.getElementById js/document id))

(hiccups/defhtml header []
  [:header
    [:h1
      [:img {:src "img/cljs-ring.svg" :alt "ClojureScript Logo"}]
      "ClojureScript Cheatsheet"]
    [:input#searchInput {:type "text" :placeholder "Search"}]])

Why ClojureScript?

  • Language-level state management
  • Build system is part of the language
  • Leverages the best of Google Closure toolchain
  • Flexible syntax: HTML, async, React components
  • Stable!
Great video on ClojureScript:
ClojureScript for Skeptics - Derek Slager

Introduction to React.js

React.js Basics

  • Declarative DOM (ie: DOM as a function of data)
  • Library handles DOM transitions
  • Component-based
  • Small, functional interface

ClojureScript + React.js

ClojureScript + React.js

  • Flexible syntax (no JSX!)
  • Immutable data by default
  • Components / functions compose naturally
